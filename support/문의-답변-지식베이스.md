# 밀비교 문의 답변 지식베이스

> 자동 답장 초안을 작성할 때 **반드시 이 문서의 사실만 근거로** 사용합니다.
> 여기 없는 내용은 추측해서 답하지 말고, 답변 초안에 "확인 후 회신드리겠습니다"로 남깁니다.

## 서비스 기본 사실 (about.html / privacy.html 기준)

- **서비스명**: 밀비교 (https://mealkit-milbigyo.netlify.app)
- **무엇**: 맛꾼·장사의신·프레시지·마이셰프 등 인기 밀키트/식자재 사이트를 **1인분 환산가격**과 **실제 후기 기반 점수**로 한 화면에서 비교·추천.
- **점수 산정**: 가격 / 맛 / 후기신뢰 3축. **가성비 종합 = 가격 45% + 맛 45% + 후기신뢰 10%**.
  - 가격 근거: 네이버쇼핑 등 대표 상품 실제 판매가
  - 맛 근거: 블로그·카페 등 실제 후기 분석
  - 후기신뢰 근거: 후기 규모·최신성
- **데이터 갱신**: 주기적으로 재수집해 갱신. 메인 상단 '최근 갱신일'에 기준 날짜 표기.
- **점수의 성격**: 공개된 후기·판매정보를 종합한 **참고용 추정치**. 구매 전 해당 사이트에서 최신 가격·리뷰 직접 확인 권장.
- **제휴 관계**: 현재 특정 업체로부터 비용 받고 순위 조정하지 않음. 향후 광고·제휴 시 '광고'/'스폰서'로 명확히 표시.
- **개인정보**: 회원가입 없음, 개인식별정보 직접 수집 안 함. 찜·별점은 브라우저 localStorage에만 저장. 광고는 구글 애드센스 사용 가능.
- **문의 주소**: milbigyo@gmail.com

## 카테고리별 답변 템플릿 (초안용)


### 1. 제휴·광고 (`문의/제휴광고`)
```
안녕하세요, 밀비교입니다.
제휴·광고 관련 문의 주셔서 감사합니다.

밀비교는 현재 특정 업체로부터 비용을 받고 순위를 조정하지 않으며,
비교 결과는 공개 데이터 기반으로 자동 산정하고 있습니다.
다만 광고·제휴 영역은 별도로 검토하고 있으며, 진행 시 해당 영역을
'광고' 또는 '스폰서'로 명확히 표시하는 것을 원칙으로 합니다.

제안 주신 내용(업체명/형태/예산 등)을 조금 더 구체적으로 알려주시면
검토 후 회신드리겠습니다.
```

### 2. 오류 제보 (`문의/오류제보`)
```
안녕하세요, 밀비교입니다.
오류를 제보해 주셔서 감사합니다. 알려주시는 정보는 사이트 품질에 큰 도움이 됩니다.

말씀하신 [항목/사이트]의 [가격/점수/링크] 부분을 확인하겠습니다.
밀비교의 가격·점수는 공개 후기·판매정보를 종합한 참고용 추정치라
실제와 차이가 생길 수 있는데, 확인 후 잘못된 부분은 빠르게 수정하겠습니다.

혹시 정확한 비교를 위해 보신 화면(상품명/가격/캡처)을 함께 주시면
더 빠르게 반영하겠습니다.
```

### 3. 개인정보 (`문의/개인정보`)
```
안녕하세요, 밀비교입니다.
개인정보 관련 문의 감사합니다.

밀비교는 회원가입이 없으며 이름·연락처 등 개인을 식별하는 정보를
직접 수집하지 않습니다. 사이트에서 설정하신 찜·별점은 서버가 아닌
이용자 브라우저(localStorage)에만 저장되며, 브라우저 데이터를 삭제하면
함께 사라집니다. 자세한 내용은 개인정보처리방침을 참고해 주세요:
https://mealkit-milbigyo.netlify.app/privacy.html

추가로 확인이 필요한 부분이 있으면 말씀해 주세요.
```

### 4. 일반 의견·제안 (`문의/일반`)
```

소중한 의견 보내주셔서 감사합니다.

말씀해 주신 내용 잘 확인했습니다. 사용자분들의 제안은 비교 항목과

업데이트하겠습니다.

앞으로도 불편하거나 추가됐으면 하는 점이 있으면 언제든 알려주세요.
```

- 답장 초안을 만들지 않습니다. 라벨만 부착하고 검토 대기로 둡니다.

## 답변 작성 규칙
- 보낸 사람 이름이 있으면 "안녕하세요 OO님," 으로 시작.
- 사실에 없는 가격·수치·약속(환불·보상 등)은 절대 적지 않음.
- 답을 확신할 수 없으면: "확인 후 다시 회신드리겠습니다."
- 톤: 정중하고 간결한 한국어 존댓말.
- 초안만 작성하고, 최종 전송은 사람이 검토 후 직접 보냄.
